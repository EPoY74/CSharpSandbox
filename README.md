# CSharpSandbox

Учебная песочница для быстрых экспериментов с C# и .NET.

## Назначение проекта

Этот проект нужен не как боевое приложение, а как безопасное место для проверки маленьких фрагментов кода.

Здесь можно быстро проверять:

- синтаксис C#;
- работу `DateTime`;
- `enum`;
- `Dictionary`;
- `List`;
- `record`;
- `class`;
- `if / else`;
- `switch`;
- `TryGetValue`;
- форматирование строк;
- работу с `CultureInfo`;
- простые алгоритмы;
- поведение предупреждений IDE и компилятора.

Главная идея:

> Не нужно каждый раз создавать новый solution/project ради проверки 5–20 строк кода.  
> Достаточно иметь одну постоянную песочницу.
